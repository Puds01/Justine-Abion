Automated Sliding Gate System Using RFID and MySQL and PHP

Our capstone project, an Automated Sliding Gate System using RFID, PHP, and MySQL, was developed to address the inefficiencies and security concerns of manual gate operations at MJ CJ Parking Rental Services. The system enhances convenience by automating access and improves security through real-time RFID-based user authentication and database-driven access management.
